# minecraft-server-c

this is a minimal Minecraft server impl in C.

it only has a working MOTD and nothing else (yet)

## Building

```sh
git clone https://github.com/p1k0chu/minecraft-server-c
cd minecraft-server-c
mkdir build
cd build
cmake ..
make
```

now you can run the executable
```sh
./main
```
